# Ridgewood Skill

A Custom Skill for Claude that helps find authentic addresses (cafés, bars, restaurants, etc.) in any city, based on a "local and non-touristy" profile calibrated on neighborhoods like Ridgewood, Bushwick, and Jackson Heights in NYC.

## What it does

Given a city or neighborhood, the skill surfaces places that feel genuinely local — the kind of spots frequented by residents, not tourists. The aesthetic and vibe benchmark is drawn from NYC's most authentic outer-borough neighborhoods.
